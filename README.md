# Les Bleus Countdown

Unofficial France men’s World Cup fan site for GitHub Pages.

## Features
- Live countdown to France v Spain
- Local kick-off time detection
- France’s route to the final
- Atlanta match weather
- Team news and opponent guide
- Five-question rotating quiz from a 200-question medium/hard bank
- Predictor, confidence meter, poll, facts, quotes and player spotlight
- Mobile and desktop responsive design

Upload all files in this folder to the root of a separate public GitHub repository, then enable GitHub Pages from the main branch and root folder.
